h1. Точка доступа к справочной информации.

h2. Позволяет получить различную справочную информацию CRM.

_Корневой адрес сервиса:_ *"/service/ref":/service/ref*
_Авторизованный доступ:_ *нет*

h3. Функции:

h4. Получить список регионов.

_URL:_ *"/service/ref/regions":/service/ref/regions*
_Поддерживаемые запросы:_ *GET*.
_Параметры:_ *нет*.
_Возвращаемое значение:_ *application/json*,  Список строк (названий регионов).

h4. Получить список марок техники (бренды).

_URL:_ *"/service/ref/motor-brands":/service/ref/motor-brands*
_Поддерживаемые запросы:_ *GET*.
_Параметры:_ *нет*.
_Возвращаемое значение:_ *application/json*,  Список строк.

h4. Получить список типов техники.

_URL:_ *"/service/ref/motor-types":/service/ref/motor-types*
_Поддерживаемые запросы:_ *GET*.
_Параметры:_ *нет*.
_Возвращаемое значение:_ *application/json*,  Список строк.
